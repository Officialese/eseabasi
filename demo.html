🏪 Eseabasi Inventory Management

A comprehensive WordPress inventory management system with real-time integration and analytics

📋 Plugin Features

📦 Import Management

Track imports for 45 different products with real-time form updates and cumulative daily tracking.

[eseabasi_import_form]

📊 Stock Management

Monitor stock levels with opening, added, used, and closing packs calculations.

[eseabasi_stock_form]

🔪 Chopped Management

Manage fruit processing from whole to prepared with pack production tracking.

[eseabasi_chopped_form]

📈 Analytics Dashboard

Beautiful analytics dashboard with modern cards showing key metrics and low stock alerts.

[eseabasi_analytics]

📋 History Tracking

Complete history with filters, pagination, and CSV export functionality.

[eseabasi_import_history]

⚙️ Admin Dashboard

Complete product and history management with functional buttons and real-time updates.

Admin Menu: Eseabasi Inventory

🔄 Integration Flow

📦
Import Form
Staff enter quantities for fruits and non-fruit items
⬇️
🍎
Fruit Processing
Fruit imports automatically update "Import Whole" in chopped form
📊
Non-Fruit Processing
Non-fruit imports automatically update "Added Packs" in stock form
⬇️
🔪
Chopped to Stock
Pack(s) gotten from chopped form update "Added Packs" in stock form
⬇️
📈
Real-time Analytics
All data reflected in analytics dashboard with low stock alerts

🎨 Design Highlights

🎯 Theme Color

Consistent #FF0000 (red) theme throughout all interfaces with modern gradients and hover effects.

📱 Mobile Responsive

Fully responsive design that works perfectly on all devices with touch-optimized interactions.

🕒 Lagos Timezone

All timestamps automatically use Africa/Lagos timezone with real-time clock updates.

✨ Real-time Updates

Calculations update instantly as you type with visual feedback and validation.

🛡️ Security & Permissions

👥 Staff Access

Limited editing permissions for specific fields only, ensuring data integrity.

👨‍💼 Admin Access

Full access to all forms, product management, and history management capabilities.

🔒 Data Validation

Comprehensive input validation prevents negative numbers and invalid entries.

💾 Auto-save

Automatic form data saving prevents data loss during long sessions.
